## Overview
This project is a **full, multi-page** responsive website for a pizzeria, built as a portfolio exercise to practice frontend fundamentals and clean code organization.

It’s also a personal project: it’s inspired by my professional background working in a pizzeria in **London**, and it’s my way to “close the loop” by combining two passions — technology and food service — into a real, presentable product.

## Tech Stack
- HTML5
- CSS3 (responsive layout)
- JavaScript (UI/DOM interactions)

## Features (examples)
- Multi-page website (not a single-page site)
- Responsive layout (mobile/desktop)
